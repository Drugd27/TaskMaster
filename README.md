# Menedżer zadań

Prosta aplikacja webowa do zarządzania zadaniami, stworzona w HTML, CSS i JavaScript z użyciem jQuery oraz DataTables.

# Funkcje
- Dodawanie i usuwanie zadań
- Oznaczanie jako ukończone / aktywne
- walidacja formularza
- Datepicker do wyboru daty
- Tabela z wyszukiwaniem i sortowaniem (DataTables)
- Eksport do CSV i PDF
- Tryb ciemny (Dark Mode)
- Animacje i interakcje UI (draggable formularz, hover, efekty)

# Użyłem: 
- HTML, CSS, JavaScript
- jQuery + jQuery UI
- wtyczki DataTables
- Bootstrap 5
